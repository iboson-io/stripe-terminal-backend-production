Available Endpoints

  • POST /connection_token - Create a connection token
  • POST /create_payment_intent - Create a payment intent
  • POST /capture_payment_intent - Capture a payment intent
  • POST /cancel_payment_intent - Cancel a payment intent
  • GET /list_locations - List Terminal locations
  • POST /create_location - Create a Terminal location

For detailed API documentation, see the Stripe Terminal docs.
